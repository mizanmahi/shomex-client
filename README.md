# SHOMEX Logistic Web App

## A full stack MERN application.

See Live: [SHOMEX](https://shomex-bd.web.app/)

### Tools I used

-  React js for frontend
-  Tailwind for design
-  React Icons
-  Hero icons
-  React Router dom
-  Firebase for authentication
-  Node js for backend
-  Mongo as Database
